Meet App

Meet app is a serverless, progressive web application that uses the Google Calendar API to provide information about events for full-stack developers.

The coding objectives of this project were to build a simple React app using a test-driven approach and to set up and configure a cloud-based server to perform user authentication and authorisation using OAuth2.0.

Technologies used: React, Jest, jest-cucumber, Enzyme, Serverless, Googleapis, Axios. AWS Lambda was used as the cloud service provider.
